If you would like to cite the SciPy tools in a paper or presentation, the following is recommended:

  • Author: Eric Jones, Travis Oliphant, Pearu Peterson and others.

  • Title: SciPy: Open Source Scientific Tools for Python

  • Year: 2001 -

  • URL: http://www.scipy.org

Here's an example of a BibTeX entry: 

@Misc{,
  author =    {Eric Jones and Travis Oliphant and Pearu Peterson and others},
  title =     {{SciPy}: Open source scientific tools for {Python}},
  year =      {2001--},
  url = "http://www.scipy.org/"
}

To cite F2PY in a paper or presentation, use the following journal article:

  • Author: Pearu Peterson
  • Title: F2PY: a tool for connecting Fortran and Python programs

  • Journal: International Journal of Computational Science and Engineering 2009 - Vol. 4, No.4 pp. 296 - 305

  • Softcopy: Authors post-peer review, accepted paper submitted for final publication by Inderscience.
